# Chatbot-Studienordnung
Ein Chatbot, der mithilfe von Retrieval Augmented Generation (RAG) Kontext zu den Studien und Prüfungsordnungen der HTWD bekommt und dazu fragen beatwortet.

Die Dokumentation dieses Projekts ist unter [docs](./docs/Dokumentation.pdf) zu finden.

Das Projekt ist in [Frontend](./frontend/README.md) und [Backend](./app/README.md) unterteilt. Das Frontend ist in Vue.js realisiert, während das Backend mittels Python und FastAPI läuft.
